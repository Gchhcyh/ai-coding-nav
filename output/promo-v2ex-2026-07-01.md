### 今天刚收录的三个新工具

**1. Superset — Agent 时代的 IDE（11.8K ⭐）**
YC 投资的桌面应用，核心功能是让你同时跑 10+ 个 AI Agent 互不干扰——每个 Agent 在独立的 Git Worktree 里工作，改完代码统一在一个 Diff 面板审查。OpenAI/Google/Microsoft 内部都在用。目前仅 macOS。
→ 刚写了 [实测体验](https://ai-coding-nav.pages.dev/notes/superset-first-look)

**2. TestSprite 3.0 — 12个并行AI帮你测Bug**
AI 写的代码让 AI 来测。12 个浏览器实例同时点击你的应用，像真实用户一样探索，发现 Bug 自动生成复现步骤。Auto-Healing 选择器能自适应 UI 变化（按钮 class 变了也能定位）。100K+ 开发者在使用。

**3. Compyle — 先问清楚再写代码**
YC 投资的 Agent，反其道而行——不像 Claude Code/Devin 那样拿了需求闷头干几个小时，它先问一堆问题直到完全理解需求，然后出规划设计文档，你确认了才开始写。适合复杂需求场景。

### 一些观察

翻了 92 个工具后发现几个有意思的趋势：
- **多 Agent 并行**：Superset、AgentFlow、BrowserAct 都在做 Agent 编排，不再是单个 AI 干活
- **CLI 为王**：最火的几个（Claude Code、Codex CLI、MiMo Code、DeepSeek TUI）都是终端工具，IDE 插件反而成了辅助
- **开源爆发**：小米 MiMo Code MIT 开源、Cohere North Mini Code Apache 2.0、Superset ELv2，大厂在把编码 Agent 能力开源化
- **AI 原生测试崛起**：AI 写的代码量爆炸，手写测试跟不上了，TestSprite、Keploy、Applitools 都在做 AI 自动测试
