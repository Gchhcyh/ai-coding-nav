# 工具分类定义

## 六大分类

### 1. ai-ide — AI IDE & 编辑器 💻
AI 原生的代码编辑器和 IDE 插件。用户在 IDE 中直接使用 AI 辅助编码。
**关键词**: IDE, 编辑器, 代码补全, inline chat, tab completion, VSCode, JetBrains, 代码生成
**典型工具**: Cursor, GitHub Copilot, Windsurf, Cody

### 2. agent-framework — Agent 开发框架 🤖
构建和编排 AI Agent 的框架和平台。用户可以搭建自定义的 AI 编程代理。
**关键词**: agent, multi-agent, workflow, orchestration, 自主编程, task planning, tool calling
**典型工具**: LangGraph, CrewAI, AutoGen, OpenManus

### 3. mcp-tools — MCP 工具 & 插件 🔌
Model Context Protocol 相关的工具、服务器和资源。
**关键词**: MCP, context protocol, tool server, plugin, integration, 上下文增强
**典型工具**: MCP Marketplaces, Atlassian MCP, Brave Search MCP

### 4. code-review — AI 代码审查 🔍
自动化代码审查和质量保障工具。AI 自动检查代码质量、安全漏洞和最佳实践。
**关键词**: code review, PR review, 代码审查, static analysis, security scanning, quality gates
**典型工具**: CodeRabbit, Copilot Code Review, Codeball

### 5. testing-qa — AI 测试工具 🧪
AI 驱动的自动化测试和 QA 工具。
**关键词**: testing, QA, test generation, 自动化测试, e2e, unit test, visual testing
**典型工具**: Applitools, Keploy, OctoMind

### 6. dev-tools — AI 开发工具 🛠️
AI 辅助的开发、部署和运维工具（不属于以上五类的）。
**关键词**: DevOps, deployment, monitoring, CLI, terminal, 运维, 部署, 监控, 文档
**典型工具**: Portkey, Galileo, Firebase Studio
