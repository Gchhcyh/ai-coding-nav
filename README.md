# AI Coding Nav

AI 编程工具全链路导航 —— 覆盖 AI IDE、Agent 框架、MCP 工具、代码审查、测试工具、开发辅助，一站找齐所有 AI 开发工具。

**78 tools · 6 categories · Updated weekly**

## Tech Stack

- **Framework:** Next.js 14 + React 18
- **Styling:** Tailwind CSS
- **Search:** Fuse.js (fuzzy search)
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Submit a Tool

Open an [Issue](https://github.com/issues) with the tool name and URL. We'll add it within 24 hours.
